特色
---
* 有着良好的初始化引导流程和界面，方便新手上手使用
* 支持一键导入关注列表中的VTB\VUP，不用手动配置冗长的房间信息
* 开播气泡提醒，让你对自己单推列表的开播状态一目了然
* 开播自动录制，让你再也不错过精彩内容
* 录制机制保证时间轴正确，让你不用忍受修复等待时间
* 支持自动文件合并和转码，并且同步到对线储存等云端
* 支持完善的鉴权API和WEB信息页，方便大家二次开发
