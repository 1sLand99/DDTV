# 弹幕录制
## 启动弹幕录制
### DDTV
在DDTV中进入`设置`界面，勾选上`录制弹幕`选项即可

### DDTVLiveRec
#### Windows
1.在``DDTVLiveRec.exe.config``文件中，将[RecordDanmu]的值改为1，也就是
```xml
<add key="RecordDanmu" value="1" />
```
2.重启DDTVLiveRec即可

#### Linux
1.在``DDTVLiveRec.exe.config``文件中，将[RecordDanmu]的值改为1，也就是
```xml
<add key="RecordDanmu" value="1" />
```
2.重启DDTVLiveRec即可
